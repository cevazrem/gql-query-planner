# gql-query-planner

`gql-query-planner` — библиотека для построения и выполнения планов GraphQL-запросов на Go.

Библиотека добавляет к обычному GraphQL-серверу отдельный слой планирования: перед выполнением запроса строится логический план, затем выбирается физический план выполнения резолверов. Планировщик может учитывать структуру GraphQL-запроса, директивы в схеме, накопленную статистику исполнения и выбирать между последовательным, параллельным и batch-выполнением.

## Возможности

- загрузка GraphQL-схемы из `embed.FS`;
- построение логического плана запроса;
- выбор физического плана выполнения;
- поддержка batchable-полей;
- поддержка cacheable-полей;
- накопление статистики исполнения резолверов;
- логирование `QUERY PLAN` и времени построения плана;
- HTTP handler для запуска endpoint GraphQL сервера.
